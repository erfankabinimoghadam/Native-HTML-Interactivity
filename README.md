# Native HTML Interactivity

This mini-project showcases how **native HTML features** can handle common interactive UI patterns without heavy JavaScript frameworks.

## Features

- Toggle content using `<details>` and `<summary>`
- Tabs using radio buttons and CSS
- Modal dialogs with `<dialog>`
- Tooltips with `<abbr>` and title
- Native form validation with HTML attributes

## Why Native HTML?

Using native HTML for interactivity improves:

- Accessibility
- Performance
- Simplicity
- Reduced JavaScript dependency

Try opening the `index.html` and explore each demo.
